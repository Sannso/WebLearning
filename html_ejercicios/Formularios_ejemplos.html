Ejemplos de las diferentes etiquetas para formularios


Etiquetas basicas para formularios:

  • Form: Envio de datos y Sin representación grafica
    • Atributos:
      • action: Requerido, Destino, URL absoluta/requerida
      • method: representa la forma de enviar la informacion que puede ser mediante GET ó POST. Si no la información se enviara por defecto con GET
      • accept-charset: Codificacion de caracteres
      • enctype: tipo de codificacion

  • Input: Texto, casillas de verificacion, casillas de opcion (radio), archivos, Texto enmascarado(password), ocultos y botones.
    • Atributos:
      • name: El atributo name es comun a todos los campos de formulario, es una cadena de texto que indica el nombre del campo al ser enviado
      • type: indica en que tipo de convertira el control, por defecto es del tipo text, pero algunos de sus tipos son:
        "text"(texto), "checkbox"(casilla de verificación), "radio"(casilla de opción *radio*), "file"(subir fichero)
        "password"(texto enmascarado), "hidden"(campo oculto), "button"(botón), "submit"(botón de enviar)
        "image"(imagen como boton de enviar), "reset"(boton de limpiar)
      • disable: inutiliza el campo para que no pueda ser utiliado:
        Ejemplo:

      • readonly: es parecido al tipo disable, pero aplicado a text permite visualizar el texto y copiarlo
        Ejemplo:

      • value: ciertos campos necesitaran tener el atributo value para completar su funcionalidad

Lo nuevo de HTML5


Se agregaron nuevas etiquetas y se eliminaron otras, a continuacion voy a mostrar algunas de las cosas nuevas:



Nuevos valores para input type


  • url: Agrega una caja de escritura para urls.
    Cuando es seleccionado en un dispositivos movil automaticamente se abrira un teclado alfanumerico
      Ejemplo:
        Ingrese aqui una URL


  • tel: Es un formato que sirve para entrar un numero de telefono.
    Cuando es seleccionado en un dispositivos movil automaticamente se abrira un teclado numerico
      Ejemplo:
        Ingrese aqui un numero de telefono


  • email: Formato que sirve para ingresar un correo electronico.
    Cuando es seleccionado en un dispositivos movil automaticamente se abrira un teclado alfanumerico
      Ejemplo:
        Ingrese aqui su correo


  • number: Formato que sirve para ingresar valores numericos, ademas que se pueden agregar restricciones.
      Ejemplo:
        Ingrese un numero de 2 o 3 cifras


  • color: Se usa para seleccionar un color que el usuario suele escoger de una paleta de colores.
      Ejemplo:
        Ingrese su color favorito


  • search: Se usa para definir campos de busqueda en forma de texto.
      Ejemplo:
        Ingrese lo que desea buscar


  • range: Se usa para contener elementos que se encuantren entre un rango en especifico en forma de deslizante.
      Ejemplo:
        Deslizante:


  • datetime: Le permite al usuario es coger un dia con la fecha completa y hora exacta.
      Ejemplo:
        Escoja su fecha (Al parecer no funciona pero datetime-local, si)



  • datetime-local: Hace lo mismo que datetime, solo que no incluye la zona horaria.
      Ejemplo:
        Ingrese una fecha


  • date: Solo permite escoger un dia/mes/año.
      Ejemplo:
        Ingrese una fecha


  • month: Permite escoger el mes de un año especificos.
      Ejemplo:
        Ingrese


  • week: Permite escoger la semana de un año especificos.
      Ejemplo:
        Ingrese


  • time: Permite escoges una hora exacta.
      Ejemplo:
        Ingrese
