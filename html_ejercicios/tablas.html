Tablas en html, ejemplos y como funcionan



    Como funciona o de que esta compuesta una tabla:
    • Las tablas estan compuestas por filas y celdas
    • La etiqueta "table" indica el inicio y final de una tabla.
    • La etiqueta "tr" indica o representa una fila de la tabla
    • La etiqueta "td" representa una celda de datos en la tabla
    • La etiqueta caption permiter agregar un titulo a la tabla
    • En las tablas es posible combinar celdas y esto se hace con los atributos colspan y rowspan
    • colgroup y col sirven para agrupar columnas


    Para agregar semantica a las tablas estan las siguiente etiquetas:
    • La etiqueta "th" (Table Header Cell) que sustituye a td para representar la cabecera
    • Tambien hay etiquetas para agrupar celdas:
      • La etiqueta "thead" agrupa las filas de la cabecera
      • La etiqueta "tbody" agrupa las filas que representan el cuerpo de los datos
      • Y la etiqueta "tfoot" agrupa las filas del pie de la tabla
